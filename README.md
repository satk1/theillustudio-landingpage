# The Illu Studio - Landing Page

A modern, light-themed landing page for The Illu Studio photography business, built with Next.js 14 and featuring elegant bokeh animations.

## Features

- Next.js 14 with App Router and TypeScript
- Modern light design with warm color palette
- Elegant bokeh bubble animation using Canvas API
- Tailwind CSS for styling
- Responsive design optimized for all devices
- SEO optimized
- Smooth scroll animations

## Design System

### Colors
- Background: `#FAFAF8` (warm white)
- Surface: `#F5F4F0`
- Primary Text: `#1A1916`
- Muted Text: `#7A7870`
- Accent: `#C8860A` (warm amber/gold)

### Typography
- Display/Headings: Cormorant Garamond (elegant serif)
- Body: Inter (clean sans-serif)

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Sections

1. Navigation - Sticky header with logo and CTA
2. Hero - Full-viewport with bokeh animation background
3. Trust Bar - Social proof statistics
4. Services - Wedding, Birthday, Business, Events
5. About - Studio story and mission
6. Testimonials - Client reviews
7. Contact - Email CTA
8. Footer - Links and copyright
